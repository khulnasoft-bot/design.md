Developers

@google/stitch-sdk

A type-safe TypeScript SDK for integrating Stitch programmatically. Works with Vercel AI SDK, Google ADK, or standalone MCP.

stitch/ai

Vercel AI SDK adapter. Returns Record<string, Tool> — drop into any generateText or streamText call.

stitch/adk

Google ADK adapter. Returns FunctionTool[] with auto-cleaned schemas for Gemini compatibility.

stitch

Core SDK with domain classes: Stitch, Project, Screen, DesignSystem. Includes MCP proxy server.
