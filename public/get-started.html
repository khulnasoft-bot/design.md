Get started with Agent Skills

Install the Stitch Design Skills, connect to the MCP server, and generate your first screen — all from your coding agent.

Stitch Design Skills are agent instructions that extend your coding agent with design superpowers — generating screens, extracting design systems, building multi-page sites, and converting designs to React components. They work with any agent that supports the Agent Skills standard: Gemini CLI, Claude Code, Cursor, or Antigravity.

1 Install the plugin suite

Run this from your project directory to install all three plugins (stitch-design, stitch-build, stitch-utilities) with 13 skills total.

Terminal
$ npx plugins add google-labs-code/stitch-skills --scope project --target claude-code

Pick your agent target

Choose the tab for your agent and copy the command:

Terminal — claude-code
$ npx plugins add google-labs-code/stitch-skills \
  --scope project --target claude-code
Install selectively

If you only need specific skills, use npx skills add google-labs-code/stitch-skills --skills stitch::generate-design. Note that many skills depend on each other — the full plugin install is the safer choice.

2 Connect the Stitch MCP server

The skills talk to Stitch through the Model Context Protocol. Follow the Stitch MCP Setup Instructions to register the server and configure your API credentials.

Once connected, your agent gains access to tools like list_projects, generate_screen_from_text, and get_screen.

Verify the connection

PROMPT
List my Stitch projects.

You should see your existing projects — or an empty list if this is your first time. If you see an error, double-check that the MCP server is running and your API key is valid.

3 Generate your first screen

Start with a rough idea. The enhance-prompt skill transforms vague concepts into structured, generation-ready prompts.

PROMPT
Enhance this prompt for Stitch: "Create a landing page for an artisanal coffee shop"

Notice the output — the skill adds a platform specification, structures the page into numbered sections, replaces vague terms with specific UI/UX keywords, and generates a design system block with color roles. This enhanced prompt produces dramatically better results than the original.

Now generate:

PROMPT
Generate a screen in my Stitch project using the enhanced prompt from the previous step.

This takes 15–30 seconds. When complete, you'll see a confirmation with the screen ID and file paths. Open the generated screenshot to see your design — a complete landing page with hero section, feature cards, navigation, and footer, all from a single text prompt.
